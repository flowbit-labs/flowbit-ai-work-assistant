# Contributing to Flowbit AI Work Assistant

Thanks for your interest in contributing to Flowbit Labs.

Our goal is to build practical, human-centered AI productivity tools that anyone can run and extend.

## How You Can Help

You can contribute by:
- Improving prompts
- Adding new assistant modes (weekly review, meeting prep, research bot)
- Improving documentation
- Fixing bugs
- Adding examples

## Getting Started

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Add clear commit messages
5. Submit a pull request

## Code Style

- Keep functions small and readable
- Write clear variable names
- Prefer clarity over cleverness
- Add comments for non-obvious logic

## Philosophy

We value:
- Simplicity over complexity  
- Practical tools over demos  
- Human-friendly UX over automation obsession  

If your change makes work calmer and clearer, it belongs here.
